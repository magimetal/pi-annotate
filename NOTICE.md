# Notice

`pi-annotate` is derived from [`nicobailon/pi-annotate`](https://github.com/nicobailon/pi-annotate).

Original developer: Nico Bailon.

This repository maintains `pi-annotate` separately under Magi Metal ownership for package structure, repository metadata, provenance, verification, and release discipline.

Attribution is intentionally preserved in shipped artifacts like `README.md`, `NOTICE.md`, and `package.json`, independent of git ancestry.

Core functionality retained from upstream source:

- `/annotate` command for starting browser annotation mode
- `annotate` tool for structured visual-annotation capture
- browser extension flow for element picking, notes, screenshots, and edit capture
- native-host bridge used for browser ↔ Pi communication
